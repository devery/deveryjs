ClientOptions :Object

This object configures how the client will connect and communicate to the Ethereum network, unless you have good reasons to change the default configurations you don't need to worry with any of these values as the will be automatically resolved contract address in your current network. Under normal circunstances you don't need to pass any of these fields.

Properties:
Name Type Description

the current web3 object, like the one injected my metamask

the accounts' addres that will execute the transactions

expects the contract address in your current network, unless you are running your own network you don't need to provide it
